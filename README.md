Built a fully functional weather web application using HTML, CSS, and JavaScript.

Features:
- City-based weather search
- Current location weather using geolocation
- 5-day weather forecast
- Dynamic weather icons and background
- Dark/Light mode toggle
- Search history using local storage
- Loading indicator for better UX

Integrated OpenWeather API to fetch real-time weather data and used modern JavaScript concepts like async/await, DOM manipulation, and local storage.
